# 💎 AgriVision Master Design System

**Concept**: Modern AI Agriculture Operating System (Agri-OS)  
**Style**: Futuristic, Calm, Trustworthy, Glassmorphism

---

## 🎨 Color Palette

### 🌑 Base Colors
- **Deep Forest (Background)**: `#07130F`
- **Surface (Secondary)**: `#0D1F18`
- **Glass Card**: `rgba(17, 38, 30, 0.72)` with `backdrop-filter: blur(18px)`

### 🧪 Brand Colors
- **Emerald (Primary)**: `#38D27D` (Used for actions, status, and AI highlights)
- **Mint (Secondary)**: `#7EF7B8` (Used for soft accents and gradients)

### ⚠️ Semantic Colors
- **Warning**: `#FFC857` (Outbreak Alerts)
- **Danger**: `#FF6B6B` (Severe Infection)
- **Info**: `#57C7FF` (General Advisory)

---

## 📐 Spacing & Layout
- **Base Unit**: `4px`
- **Container**: `1280px` max-width
- **Section Padding**: `clamp(2rem, 8vw, 6rem)`
- **Border Radius**: 
  - Components: `12px` (Medium)
  - Cards: `24px` (Large)
  - Buttons: `100px` (Pill)

---

## ✍️ Typography
- **Primary Font**: `Geist` (Modern, monospace-adjacent for data)
- **Secondary Font**: `Inter` (Readability for long text)
- **H1 Header**: `700` weight, `-0.02em` tracking
- **Body Text**: `400` weight, `1.6` line-height

---

## 🧊 Visual Effects
- **Gradients**: 
  - `linear-gradient(135deg, #07130F 0%, #0F2D20 40%, #174B33 100%)`
- **Shadows**: Soft, deep green-tinted shadows for depth (`rgba(0, 0, 0, 0.4)`)
- **Borders**: Subtle `1px` solid `rgba(255, 255, 255, 0.08)` for glass cards.

---

## 💠 Component Principles
1. **AI-First**: Every interactive element should feel "smart" (hover glows, subtle micro-animations).
2. **Transparency**: Use layered transparency to show depth.
3. **Calmness**: Avoid clutter; use high whitespace to reduce cognitive load for farmers.
